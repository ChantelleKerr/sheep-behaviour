# sheep-behaviour

## Create Virtual Environment

```
python3 -m venv venv
```

or

```
python -m venv venv
```

## Activate Virtual Environment

macOS

```
source venv/bin/activate
```

windows

```
venv\Scripts\activate
```

## Install requirements

```
pip install -r requirements.txt
```

## Run Program

```
python3 main.py or python main.py
```
